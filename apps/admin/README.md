# Admin Dashboard Application

This application handles the administrative tasks for the portfolio platform, offering a unified UI to manage content and view analytics.

## Folder Structure
- `app/`: Next.js App Router providing routing for dashboard pages.
- `components/`: UI components and shared UI assets.
- `hooks/`: Specialized logic separating UI state and side effects.
- `utils/`: Common pure functions across the admin app.

## Development
Run `pnpm dev` inside this directory or from the root via `pnpm --filter admin dev`.
